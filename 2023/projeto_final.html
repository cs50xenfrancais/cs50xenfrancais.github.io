Projeto Final

O clímax deste curso é o seu projeto final. O projeto final é a sua oportunidade de pegar os conhecimentos adquiridos em programação e desenvolver seu próprio software. Contanto que seu projeto se inspire nas lições deste curso, a natureza do seu projeto é totalmente sua escolha. Você pode implementar seu projeto em qualquer linguagem (ou linguagens). Você também pode usar outras plataformas além do CS50 IDE. Tudo o que nós pedimos é que você crie algo do seu interesse, que resolva um problema real, que você impacte sua comunidade, ou que você mude algo no mundo. Esforce-se para criar algo cuja utilidade vá além desde curso.

Ideias

  • um aplicativo web usando JavaScript, Python, e SQL, baseado em parte no código fonte da especialização em desenvolvimento web
  • um jogo usando Lua com LÖVE
  • uma extensão Chrome usando JavaScript
  • um programa de linha de comando usando a lingaugem C
  • um aplicativo baseado em hardware programado para algum dispositivo
  • …

Como Enviar

Passo 1 de 2

Crie um arquivo de texto README.md que explique seu projeto e salve ele em uma pasta chamada project, no seu diretório ~/. Note que o código fonte do seu projeto não precisa ser enviado, apenas este arquivo README.md

Execute o código abaixo dentro do seu diretório ~/project, acessando sua conta GitHub informando seu nome de usuário e sua senha. Por medidas de segurança, você verá asteríscos ao invés dos caracteres reais da sua senha.

submit50 cs50/problems/2020/x/project

Passo 1 de 2

Envie um vídeo curto (que não tenha mais de 2 minutos de duração) onde você apresenta seu projeto para o mundo, com slides, capturas de telas, narração, e/ou testagem do aplicativo/software. Seu vídeo deve incluir o título do seu projeto, seu nome, sua cidade e seu país, e qualquer outro detalhe que você gostaria de comunicar para seus espectadores. Leia este link para dicas de como fazer uma captura de tela do seu computador, mas você também pode usar uma câmera de verdade, se quiser. Envie seu vídeo para o YouTube e anote o URL do vídeo. Não tem problema se você definir o vídeo como "não listado", mas não define o vídeo como "privado."

Quando você estiver pronto para enviar seu vídeo, responda este formulário!

É isso! Seu projeto será avaliado em alguns minutos. Se você não ver nenhum resultado no seu boletim, tente reenviar (executando o comando submit50 acima) contendo somente seu arquivo README.md desta vez. Não é necessário reenviar o formulário.
